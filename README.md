# ruby-the-noises

[ザ・ノイジーズ](http://www.enterjam.com/?cid=3)のポッドキャスト用Rubyクライアント。

## Installation

```
$ gem instal the-noises
```

## Usage

download an entry

```
$ the-noisies download 300
Title: 第300回 本日のお題：なんと飛び級で300回目を迎えちゃいました！テヘ！300回記念回「アイドル八犬伝について語ろう」について語ろう！

Time: 00:00:33 <=================================================================================> 100% Progress
```
